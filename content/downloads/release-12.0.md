dCache 12.0 is a Feature Release introducing following highlights:
- SciTag support
- Retry stage request on `pool up`

#### Incompatibilities
- Removed SRM and SRM server functionality.
- Removed NIS support

_Starting with dcache 10.1.0 Java 17 is required for runtime_
